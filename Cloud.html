Cloud Computing

"Cloud" is essentially a network. Users can access resources on the cloud at any time. As long as you pay by usage,
the "cloud" can be seen as infinitely scalable.

Cloud computing is not a new network technology, but a new network application concept. So that everyone who
uses the Cloud can use the huge computing resources and data center on the network.

The most common cloud computing application in our life should
be cloud storage. Cloud storage not only liberates users' insufficient
local storage, but also makes data more secure and not easy to lose.
In addition, cloud computing is also used in the work. By using cloud
computing, the concept of office is weakened. Maybe in the future,
we can work at home every day.
